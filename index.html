How it works
1
Pick a soul from the souls/ directory on GitHub
2
Copy the SOUL.md into your OpenClaw workspace
3
Restart your agent session
4
Your agent now thinks, speaks, and prioritises like your chosen character

Add a soul

Fork the repo, create your souls/character-name/SOUL.md using the template, keep the name plausibly deniable, and open a PR. The soul should be genuinely useful as an agent personality — not just a joke.
