# famous-souls

> Drop-in `SOUL.md` files for [OpenClaw](https://openclaw.ai) — personalities inspired by legendary figures.  
> Not the real ones. You'll figure it out.

---

## What is this?

OpenClaw agents are shaped by a `SOUL.md` file — it defines how an agent thinks, communicates, and prioritises. Swap the file, swap the character.

This repo is a community collection of `SOUL.md` files built around archetypes inspired by famous (and infamous) people. Each file is ready to drop into your OpenClaw workspace.

---

## How to use

1. Pick a soul from the `souls/` directory
2. Copy the `SOUL.md` into your OpenClaw workspace (replace or rename your existing one)
3. Restart your agent session
4. Enjoy

## Contributing

Want to add a soul?

1. Fork the repo
2. Create `souls/your-character-name/SOUL.md` (use the [template](template/SOUL.md))
3. Keep the name plausibly deniable
4. Open a PR

**Rules:**
- No real names — ever
- Keep it fun, not mean-spirited
- The soul should be genuinely useful as an agent personality, not just a joke
- Stick to the SOUL.md structure
